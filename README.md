# Plant Disease Detection Pro 🌿

Plant Disease Detection Pro is an advanced, AI-powered web application that helps farmers, gardeners, and plant enthusiasts identify plant diseases from images of plant leaves. The application uses a trained deep learning model to provide high-precision diagnostics and offers expert recommendations for treatment.

## Features ✨

*   **High-Precision AI Diagnostics:** Upload a photo of a leaf or take one using your camera to instantly identify potential diseases.
*   **Comprehensive Disease Database:** Detailed symptoms and management strategies for various plant diseases affecting crops like Pepper, Potato, and Tomato.
*   **Environment Dashboard:** Simulated real-time metrics for Temperature and Humidity with automated risk alerts for fungal growth.
*   **Expert Recommendations & Reports:** Access treatment protocols and download diagnosis reports.
*   **Find Nearby Experts:** Connect with plant health experts in your area through our local doctor directory.
*   **Premium UI/UX:** A modern, glassmorphism-inspired design for a smooth and intuitive user experience.

## Tech Stack 🛠️

*   **Frontend:** [Streamlit](https://streamlit.io/)
*   **Backend & Machine Learning:** TensorFlow, Keras, NumPy, Pandas
*   **Image Processing:** Pillow (PIL)

## Installation & Setup 🚀

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/VedantVH/Plant-Disease-Detection.git
    cd Plant-Disease-Detection
    ```

2.  **Install dependencies:**
    Ensure you have Python installed, then run:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the application:**
    ```bash
    streamlit run streamlit_app.py
    ```

## Usage 💡

1.  Launch the application using the command above.
2.  Upload a clear image of a plant leaf or use your device's camera to capture a photo.
3.  Wait for the AI to analyze the leaf patterns.
4.  Review the diagnosis report, confidence score, and recommended treatment protocols.
5.  Use the "Find Nearby Plant Health Experts" section to locate agricultural experts in your vicinity.
